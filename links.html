Programming

Tutorials

Reading a significant portion of any of these will give you the basics required to start programming. When it comes to programming books and tutorials, what you read will only matter if the concepts that you've learned help you in writing code.

  • Structure and Interpretation of Computer Programs (SICP)

    • An introduction to computation and programming. Racket is a good choice for running the example code and doing the exercises from the book.

  • The Python Tutorial

    • A nice tutorial to start programming with.

  • Learn you a Haskell for great good

    • Learn the basic functional programming in Haskell.

  • Eloquent Javascript

    • A modern introduction to Javascript.

  • Why's poignant guide to Ruby

    • gsathya refuses to suggest a suitable description for this tutorial.

  • Practical Common Lisp

    • A practical introduction to Common Lisp.

  • LLVM Tutorial

Newer programming languages

  • Rust

    • A systems programming language combining that's currently being developed at Mozilla. It aims to ultimately be used for the things C++ is being used for at the moment.

  • Shen

    • Shen, draws its principles from functional programming and the ancient Lisp tradition. It has some interesting features.

Tools

Text editors

Knowing how to efficiently use a text editor is one of the most useful secondary skills of any programmer.

  • Vim

    • A modal text editor that has nice keybindings and an emphasis on speed.

  • Emacs

    • An operating system containing (amongst many other things) a file manager, calculator, games, package manager and even a text editor. It can be extended in trivial, and extremely non-trivial ways with Emacs Lisp.

  • Notepad++

    • An easy to use, feature-laden text editor.

  • LightTable

    • A new interactive IDE written in ClojureScript.

Version Control

A Version Control System (VCS) is used to track changes in files. It makes distributed collaborative development less painful.

  • Mercurial

    • Mercurial is supposedly easier to learn and use when compared to Git. This nicely illustrated tutorial covers the fundamental version control concepts.

  • Pro Git

    • A solid introduction to the powerful Git version control system.

Command-line

  • zsh

    • A shell with nice completions.

  • tmux

    • Opening multiple terminal emulator windows gets unmanageable pretty quickly. This terminal multiplexer can be used with a tiling window manager for a killer combination.

Window Managers

Tiling

Tiling window managers are useful in a typical programming session when you have to repeatedly switch between your text editor, terminal emulator, web browser, and any other programs you may have running.

  • xmonad

    • One of the more widely used Haskell programs.

  • awesome

    • Another good tiling window manager.

Graphics

  • Inkscape

    • The best open source SVG editor there is.

Miscellaneous

  • Pandoc

Easily converts between a great variety of markup formats.

References

Learning to use references is one of the essential skills of a programmer. They are large and intimidating to begin with, but with time you will learn how indispensable they are.

Language References

  • C++ Reference

    • Handy quick reference while writing C++ code. It can be downloaded to be used offline.

  • C++ Reference

    • Another reference on C++. It has clear examples to help you learn how to use the standard library.

  • C++ FAQ

    • Fairly large collection of C++ FAQs and their answers.

  • C++ FQA

    • C++ Frequently Questioned Answers.

  • Simplified Common Lisp Reference
  • Common Lisp HyperSpec™

    • The entire ANSI CL standard in HTML.

Protocols, Specifications

  • RFC

    • Freely available specifications for various internet-related protocols.

Programming Exercises

  • Project Euler

    • A good source of many exercise problems. Many of the problems here require some mathematical (and programming) knowledge.

  • Programming Praxis

    • Some practical programming problems in addtition to the sorts of problems found at Project Euler

Operating Systems

Any operating system that you are productive in, and happy with, is fine. If that is not the case, however, these are some of our recommendations.

  • Debian

    • neena says that it is a decent operating system. A widely used, stable Linux distribution.

  • Arch Linux

    • A Linux distribution with an emphasis on simplicity, configurability and having the latest software packages. Rolling releases keep your system at the bleeding edge.

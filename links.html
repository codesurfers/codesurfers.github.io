Tools

Text editors

Knowing how to efficiently use a text editor is one of the most useful secondary skills of any programmer.

  • Vim

A modal text editor that has nice keybindings. Its emphasis is on speed.

  • Emacs

A text editor that is part of an operating system containing a file-manager, calculator, games and even a package manager. It can be extended in trivial and extremely non-trivial ways with Emacs Lisp.

  • Notepad++

A feature-laden, easy to use text editor.

  • LightTable

A new interactive IDE.

Version Control

A Version Control System (VCS) is used to track changes in files. It also makes distributed collaborative development possible.

  • Mercurial

Mercurial is supposedly easier to learn and use when compared to Git. This tutorial covers the basics of version control using Mercurial.

  • Pro Git

A solid introduction to the powerful Git version control system.

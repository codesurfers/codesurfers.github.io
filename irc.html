#codesurfers

#codesurfers is an IRC channel in the OFTC network.

We mostly talk about programming and free software, so if you like talking about such things, we welcome you to #codesurfers.

Join us!

You'll need to use an IRC client to join the channel. Some commonly used IRC clients are are irssi, weechat and XChat.

It's good practice to register your IRC nickname once you start using IRC regularly.

irssi can be used in conjunction with a GNU Screen session to keep it running indefinitely on a VPS.

We are usually the most chatty between 2300-0300.
